## Prerequisites
- **Subscription**: Non-production Azure lab subscription with owner/contributor permissions
- **RBAC permissions**: Contributor or Owner role (or specific roles noted per lab)
- **Tools installed**:
  - Azure Portal (browser)
  - Azure CLI (`az` command)
  - PowerShell with Az module (`Connect-AzAccount`)
  - (Optional) Bicep CLI for IaC labs
- **Prior knowledge**: Recommended: AZ-104 Prerequisites module (basic Azure concepts, cloud terminology)

## Accessibility & Support
- **Portal labs**: All labs support keyboard navigation and high-contrast mode
- **CLI labs**: All commands can output JSON (use `--output json`) for screenreader compatibility
- **Bicep labs**: Automated deployment alternative available for students with time constraints
- **Instructor support**: Estimated response time: within 1 business day (see cohort guide for escalation)

## Self-Assessment & Learning Path
1. **Before starting**: Take the pre-module knowledge check ([knowledge-check-pre.md](knowledge-check-pre.md)) to identify prerequisite gaps
2. **During module**: Follow labs in sequence (Beginner → Intermediate → Advanced based on experience level)
3. **After module**: Take the post-module knowledge check ([knowledge-check-post.md](knowledge-check-post.md)) to verify learning
