# M01 - Virtual Machine Foundations

## Module Overview
| Item | Details |
|------|----------|
| **Duration** | Beginner: ~60 min \| Intermediate: ~75 min \| Advanced: ~95 min |
| **Prerequisites** | AZ-104 prerequisites recommended |
| **Difficulty** | All tiers available |
| **Labs** | Beginner + Intermediate + Advanced alignment |

## Learning Outcomes
You will:
- Create and configure Azure virtual machines
- Select VM sizes, disks, and authentication models for workload needs
- Configure VM networking, NSG association, and access methods
- Validate VM provisioning and baseline operations with Portal, CLI, and PowerShell

See [objectives.md](objectives.md) for AZ-104 mapping.

## Prerequisites & Permissions
- Non-production lab subscription
- Contributor or Owner at target resource group scope
- Azure Portal, Azure CLI, and PowerShell Az installed

## Accessibility & Support
- Portal path and CLI/PowerShell parity are provided in module labs
- Use JSON output for screenreader-friendly command verification

## Self-Assessment Path
1. Complete pre-check (if enabled)
2. Run module labs (Portal, CLI, PowerShell, Bicep where applicable)
3. Execute validation script
4. Complete post-check (if enabled)

## Deliverables (Definition of Done)
- VM deployment and configuration evidence captured
- Required settings verified (size, disk, network, login path)
- Validation script returns `PASS`
- References are official Microsoft Learn/Docs
