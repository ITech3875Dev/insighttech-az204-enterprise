# M04 - Compute Security and Operations

## Module Overview
| Item | Details |
|------|----------|
| **Duration** | Beginner: ~55 min \| Intermediate: ~70 min \| Advanced: ~100 min |
| **Prerequisites** | M01 through M03 recommended |
| **Difficulty** | Beginner and Intermediate core, Advanced extensions |
| **Labs** | Beginner + Intermediate + Advanced alignment |

## Learning Outcomes
You will:
- Configure VM security baseline controls (JIT, Defender recommendations, identity access)
- Configure backup and recovery posture for compute resources
- Implement monitoring and alerting for compute availability and health
- Validate security and operational controls with remediation evidence

See [objectives.md](objectives.md) for AZ-104 mapping.

## Prerequisites & Permissions
- Non-production lab subscription
- Contributor role and policy read access at target scope
- Azure Portal, Azure CLI, and PowerShell Az installed

## Accessibility & Support
- All tasks have equivalent command-based verification paths
- Output artifacts can be collected in table, CSV, or JSON format

## Self-Assessment Path
1. Confirm security and operations prerequisites
2. Complete security and governance labs
3. Run validation script and review failures
4. Capture remediation actions and re-validate

## Deliverables (Definition of Done)
- Security and operations baseline evidence captured
- Monitoring, alerting, and recovery findings documented
- Validation script returns `PASS`
- References are official Microsoft Learn/Docs
