# M03 - Load Balancing and Name Resolution

## Module Overview
| Item | Details |
|------|----------|
| **Duration** | Beginner: ~60 min \| Intermediate: ~80 min \| Advanced: ~105 min |
| **Prerequisites** | M01 and M02 completion recommended |
| **Difficulty** | All tiers available |
| **Labs** | Beginner + Intermediate + Advanced alignment |

## Learning Outcomes
You will:
- Configure Azure Load Balancer components and rules
- Compare internal/external balancing patterns and health probes
- Configure Azure DNS zones and record management
- Validate traffic flow and name resolution outcomes with evidence

See [objectives.md](objectives.md) for AZ-104 mapping.

## Prerequisites & Permissions
- Non-production lab subscription
- Contributor role at resource group scope
- Permission to create load balancer and DNS resources

## Accessibility & Support
- Equivalent instructions are available across Portal, CLI, and PowerShell
- Validation and evidence commands support JSON output

## Self-Assessment Path
1. Confirm load balancing and DNS prerequisites
2. Execute labs in sequence
3. Validate sync/movement outcomes
4. Capture rollback and troubleshooting notes

## Deliverables (Definition of Done)
- Load balancer and DNS configuration evidence captured
- Traffic and name-resolution verification documented with outcomes
- Validation script returns `PASS`
- References are official Microsoft Learn/Docs
