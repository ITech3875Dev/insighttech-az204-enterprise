# Lab 01 (Beginner): Function HTTP API

## Scenario
Create a minimal HTTP-triggered Azure Function that returns a structured JSON response for a simple service endpoint.

## Prerequisites
- Azure subscription access.
- Azure CLI and Functions Core Tools installed.
- Resource group naming standard selected.

## Tasks
1. Create a resource group for the lab.
2. Create a Function App and storage dependency.
3. Implement an HTTP trigger endpoint.
4. Deploy and test the endpoint.
5. Capture response evidence for validation.

## Validation Checks
- Function app is running.
- HTTP endpoint returns a successful response.
- Log output is visible for invocation.

## Cleanup
Delete all resource-group assets created by this lab.
