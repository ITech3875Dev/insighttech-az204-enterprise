# AZ-204 LP01: Develop Azure Compute Solutions

## Overview
This learning path focuses on building and deploying compute-driven application components for AZ-204.

## Outcomes
- Build and deploy Azure Functions and App Service workloads.
- Configure app settings and managed identity integration.
- Implement deployment workflows and safe release patterns.
- Validate compute workloads using repeatable checks.
