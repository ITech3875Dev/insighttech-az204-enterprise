# Module M01: App Service and Functions

## Learning Objectives
- Build HTTP-triggered Azure Functions for application endpoints.
- Deploy and configure Azure App Service for web/API workloads.
- Use deployment slots for safe release patterns.
- Apply app configuration and identity best practices.

## Suggested Sequence
1. Review compute service selection guidance.
2. Build a simple HTTP function endpoint.
3. Deploy an App Service app and configure settings.
4. Use a staging slot and perform a controlled swap.

## Deliverables
- Deployed function endpoint.
- Deployed web app with staging slot.
- Configuration notes and validation evidence.
