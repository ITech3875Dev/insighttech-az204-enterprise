# Grading Rubrics

## Module (100 points)
- 25: Objectives mapped to skills measured
- 20: Portal lab correctness (Beginner: click-by-click)
- 15: CLI lab correctness
- 15: PowerShell lab correctness
- 10: Bicep correctness (where applicable)
- 10: Validation quality (clear PASS/FAIL, meaningful assertions)
- 5: References quality (official, relevant)

## Labs (Beginner/Intermediate/Advanced)
- Beginner: clarity, step-by-step, verification after each task
- Intermediate: fewer screenshots, more checkpoints and troubleshooting
- Advanced: scenario-based, acceptance criteria, architecture reasoning
