# Cohort Guide (Students)

## How you work
1. Pull latest `main`
2. Create a branch: `lp01-m02-policy-yourname`
3. Copy the module template from `shared/templates/module/`
4. Fill in content using the standards
5. Run validation scripts
6. Open a PR and include evidence

## Definition of Done
- Portal steps are reproducible (Beginner track requires click-by-click)
- CLI and PowerShell steps execute successfully
- Bicep compiles where included
- Validation script returns PASS
- References are official (Microsoft Learn/Docs)
- No secrets committed

## Note for macOS/Linux students (CLI scripts)
If you run the shared bash scripts, ensure they are executable:
```bash
chmod +x shared/scripts/cli/validation/tag-compliance-report.sh
```
