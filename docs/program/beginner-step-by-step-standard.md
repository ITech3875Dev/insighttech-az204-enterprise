# Beginner Track Standard (Enterprise)

Beginner labs must be reproducible by a student with minimal Azure experience.

## Required components
1. Portal walkthrough (click-by-click)
2. Az CLI walkthrough (copy/paste)
3. PowerShell walkthrough (copy/paste)
4. Bicep walkthrough (where applicable)
5. Verification steps after each task
6. Troubleshooting: common errors + fixes

## Formatting rules
- Every task includes: Goal → Portal → CLI → PowerShell → (Bicep) → Verify → Troubleshoot
- Always set subscription context explicitly
- Use repo naming and required tags
